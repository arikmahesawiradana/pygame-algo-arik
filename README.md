### ğŸš€ Cara Mengunduh dan Menjalankan (Eksekusi)

Ikuti langkah-langkah di bawah ini untuk menjalankan **Pymaze** di mesin lokal Anda.

#### 1\. Persyaratan (Prasyarat)

Pastikan Anda telah menginstal **Python 3.x** di sistem Anda.

#### 2\. Kloning Repositori

Buka terminal atau Command Prompt Anda dan jalankan perintah berikut:

```bash
git clone https://github.com/NamaPenggunaAnda/NamaRepositoriAnda.git
cd NamaRepositoriAnda
```

*(Ganti `NamaPenggunaAnda/NamaRepositoriAnda` dengan detail repositori Anda yang sebenarnya.)*

#### 3\. Instalasi Pygame

Anda perlu menginstal pustaka Pygame. Gunakan **pip**, pengelola paket Python, untuk menginstalnya:

```bash
pip install pygame
```

#### 4\. Menjalankan Game

Setelah Pygame terinstal, Anda dapat menjalankan game dengan menjalankan file utama (misalnya, `main.py`) menggunakan Python:

```bash
python main.py
```

Setelah perintah ini dijalankan, jendela game Pymaze akan terbuka, dan Anda bisa langsung bermain\!
